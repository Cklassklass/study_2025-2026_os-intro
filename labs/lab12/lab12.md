# Теоретическое введение

**Командный процессор (shell)** — программа, обеспечивающая взаимодействие пользователя с операционной системой. В UNIX/Linux распространены следующие оболочки:

- **Bourne shell (sh)** — базовая оболочка UNIX;

- **C shell (csh)** — с синтаксисом, похожим на язык C;

- **Korn shell (ksh)** — сочетает возможности sh и csh;

- **Bash (Bourne Again Shell)** — наиболее популярная современная оболочка.

**Командный файл (скрипт)** — текстовый файл с последовательностью команд для shell, который можно выполнить как единую программу.
