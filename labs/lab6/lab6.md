# Теоретическое введение

Командная строка (терминал) Linux — это

интерфейс для управления операционной системой с помощью текстовых команд. Они позволяют эффективно работать с файлами, процессами и настройками, используя базовые команды (например, `cd`, `ls`, `mkdir`, `rm`, `sudo`). Терминал обычно открывается через меню приложений или по сочетанию клавиш `Ctrl+Alt+T`
